Show timing differences between sequential vs concurrent. Mobile parallel: Swift `async let`, Kotlin `awaitAll()`, Dart `Future.wait()`.     3. `03-semaphores-limiting.md` — Controlling concurrency: `asyncio.Semaphore(n)` to limit parallel operations, `async with semaphore:` pattern (never manual acquire/release), practical use cases (rate limiting API calls, limiting DB connections, controlling file I/O). Mobile parallel: `DispatchSemaphore` (Swift), `Semaphore` in Kotlin coroutines, custom limiters in Dart. Show before/after: 100 concurrent requests overwhelming an API vs 10-at-a-time with semaphore.

    4. `04-async-context-managers.md` — Resource management with async: `__aenter__`/`__aexit__`, `@asynccontextmanager` decorator, practical examples (database connections, HTTP client sessions, file handles). Show building a custom async API client class.

    5. `05-async-generators.md` — Streaming data with async: `async for`, `async yield`, `AsyncIterator` protocol, practical use cases (paginated API responses, streaming database results, processing large files in chunks). Mobile parallel: Swift `AsyncSequence`, Kotlin `Flow`, Dart `Stream`. Show building a paginated API fetcher that yields pages one at a time.

    6. `06-exception-handling.md` — Error handling in async code: exceptions in coroutines, `return_exceptions=True` in gather, `ExceptionGroup` and `except*` syntax (Python 3.11+) with TaskGroup, cancellation handling (`asyncio.CancelledError`), timeout with `asyncio.wait_for()` and `asyncio.timeout()`. Mobile parallel: Swift throwing functions in tasks, Kotlin `supervisorScope` vs `coroutineScope`, Dart zone error handling. Show the difference between gather (partial results) and TaskGroup (all-or-nothing with exception groups).
