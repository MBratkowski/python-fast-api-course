# Backend Development Mastery — Course Content

## What This Is

A hands-on course for mobile developers transitioning to backend development with Python and FastAPI. 25 modules covering foundations through production deployment, each with theory explanations, coding exercises, and mini-projects. Targeted at developers who already know programming but are new to backend/Python.

## Core Value

Every module delivers practical, hands-on learning content that a mobile developer can work through independently — theory with mobile-context analogies, exercises with built-in tests, and self-contained projects.

## Requirements

### Validated

- Module 001 (Python Backend Quickstart): Complete with theory (5 files), exercises (3 files), project (1 README)
- All 25 module README.md files exist with topic outlines

### Active

### Out of Scope

- Solution files for exercises — learners fill in stubs themselves
- Progressive/cumulative project builds across modules — each project stands alone
- Video content or interactive tutorials — text and code only
- Deployment infrastructure or live API hosting

## Context

- Target audience: Mobile developers (iOS/Android/Flutter/React Native) learning backend
- Module 001 establishes the content pattern: mobile-dev analogies, practical examples, pytest-based exercises
- Each module README already outlines topics and theory structure
- Tech stack: Python 3.12+, FastAPI, PostgreSQL, SQLAlchemy, Pydantic v2, JWT, Redis, Docker, pytest

## Constraints

- **Content pattern**: Must follow module 001 structure (theory/, exercises/, project/)
- **Exercise format**: Python files with TODO stubs and inline pytest tests
- **Theory format**: Markdown with "Why This Matters" mobile-dev framing, code examples, "Key Takeaways"
- **Project format**: README.md with requirements, starter template, success criteria
